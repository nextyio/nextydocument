Window Version¶

Data directory of Nexty¶

  • Create new folder C:\gonex\bin

  • Put gonex.exe file to the folder eg: C:\gonex\bin

Config environment¶

  1. Go to System

  2. Click Advanced System Settings

  3. Environment Variables

  4. On tab System Variables, click Edit

  5. On System Variable window, click New, insert C:\gonex \bin

  6. Click OK, OK and OK to accept new gonex path.
